---

## ⚙️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica com `<header>`, `<main>`, `<section>`, `<footer>`
- **CSS3 puro** — sem frameworks; variáveis CSS, grid, flexbox, media queries, animações
- **JavaScript vanilla** — sem frameworks; Intersection Observer, localStorage, validação de formulário

> ⚠️ Nenhum framework de CSS (Bootstrap, Tailwind etc.) ou de JS (React, jQuery etc.) foi utilizado, conforme exigido pela atividade.

## ✅ Requisitos da Atividade Atendidos

- [x] Página única com navegação por âncoras (Single Page Website)
- [x] Menu fixo no topo visível em todas as seções
- [x] Mínimo de 4 seções interligadas pelo menu
- [x] Seção **Sobre Mim** com informações pessoais e hobbies
- [x] Seção **Formação** com histórico acadêmico e técnico
- [x] Seção **Portfólio** com links para projetos realizados
- [x] Seção **Contato** com formulário (nome, e-mail, mensagem)
- [x] Validação de formulário em JavaScript (campos obrigatórios + formato de e-mail)
- [x] Simulação de envio com mensagem de confirmação visual
- [x] Menu responsivo (hamburger) para dispositivos móveis
- [x] Alternância de tema claro/escuro via JavaScript (☀ / ☽)
- [x] Scroll-reveal com Intersection Observer API
- [x] Código comentado nos pontos de relevância
- [x] `<!DOCTYPE html>` HTML5
- [x] Sem uso de frameworks de CSS ou JavaScript
- [x] Publicado no GitHub Pages
- [x] Repositório público no GitHub

## 🚀 Como Rodar Localmente

```bash
# Clone o repositório
git clone https://github.com/eldermf/portfolio.git

# Entre na pasta
cd portfolio

# Abra no navegador (qualquer um dos dois)
xdg-open index.html        # Linux
open index.html            # macOS
```

Ou use a extensão **Live Server** do VS Code para servir localmente.
